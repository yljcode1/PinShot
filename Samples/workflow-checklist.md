# PinShot Manual Checklist

- [ ] Launch PinShot and confirm menu bar icon appears.
- [ ] Trigger global hotkey (`Command + Shift + 2` by default) and ensure selection overlay shows.
- [ ] Pin result, adjust opacity, and verify dragging/zooming works.
- [ ] Run OCR → compare output with `ocr-demo.txt`.
- [ ] Trigger automatic translation (if enabled) and confirm language toggle.
- [ ] Reopen latest capture from history menu.
- [ ] Quit PinShot via menu and relaunch to confirm persisted settings.
