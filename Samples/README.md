# Samples

Prepared examples to help validate OCR, translation and pinning flows without capturing new screenshots each time.

## Included files

- `ocr-demo.txt` – multi-language text that mimics OCR output. Use it to test translation logic or annotation tools by pasting into a pin.
- `workflow-checklist.md` – quick checklist for manual regression passes (global hotkey, pin controls, OCR, translation).

## Usage tips

1. Open a pin, choose the **文字** tool, and paste the sample text for layout checks.
2. For OCR verification, copy `ocr-demo.txt` into an image (e.g., Preview → Text Tool) and run PinShot’s OCR to ensure the result matches the file.
3. Update or extend these samples as new features are added so the team has ready-made fixtures for manual or automated tests.
