# PinShot 0.3.3

## 更新内容

- 新增首次启动引导，支持屏幕录制、辅助功能、开机启动与快捷键配置，并可跳过
- 新增历史搜索与筛选，可按 OCR、已翻译、已标注快速过滤
- 新增导出增强，支持 `PNG`、`JPEG`、`OCR Text`、`Translated Text` 与 `Capture Package`
- 优化贴图工具栏与 OCR/翻译面板，支持更快复制、保存和继续编辑
- 优化小图贴图体验，移除遮挡画面的悬浮提示与状态标签

## 验证

- `swift build`
- `swift run PinShot --all-checks`
- 本地安装版已替换并完成启动验证
